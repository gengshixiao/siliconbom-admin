# 硅宝运营平台

智能体产品运营后台系统

## 项目说明

这是一个纯前端运营后台系统，用于查看硅宝智能体产品的使用数据、核心运行指标、用户反馈信息等内容。

## 快速开始

1. 直接在浏览器中打开 `index.html`
2. 使用默认账号密码登录：
   - 账号：`admin`
   - 密码：`admin123`
3. 登录后即可使用系统各项功能

## 功能模块

### 运营管理
- **用户运营看板** - 查看用户数据指标和趋势
- **智能体运营看板** - 查看智能体运行数据和性能指标
- **BOM及知识库运营看板** - 查看BOM和知识库的使用情况

### 用户管理
- **用户列表** - 查看用户信息、会话记录等
- **产品反馈** - 查看用户反馈，支持标签筛选

## 技术特点

- 纯前端实现，无需后端服务
- 无外部依赖，可直接在浏览器中运行
- 使用深灰色调设计，简约清晰
- 低饱和度配色方案，视觉舒适
- 响应式设计，适配不同屏幕

## 文件结构

```
硅宝后台/
```

## 注意事项

- 所有数据均为模拟数据，仅用于演示
- 登录状态使用 sessionStorage 管理
- 系统为纯前端实现，数据不会持久化保存
